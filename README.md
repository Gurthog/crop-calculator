# Crop Calculator
Crop Calculator is a tool for players of Stardew Valley which shows the value of crops across a range of game states.
This app is inspired by [Thorinairs' "Stardew Profits"](https://github.com/Thorinair/Stardew-Profits) tool, and will be built as practice.

Thoughts and notes are recorded within the [notes](notes/README.md) directory.
